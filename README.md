#SIP_NET

网上找到的一款使用C# .NET开发的SIP项目。测试可编译通过运行，特此分享一下。

 **需要安装各种SIP/H323软交换、ASTGO、VOS2009、VOS3000的可以访问 [www.51voip.org](http://www.51voip.org)获取各种软交换安装包、安装教程、API文档资料、注册激活等资源。**
